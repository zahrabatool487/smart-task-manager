# Smart Task Manager 📝

A responsive React.js application built to manage daily tasks efficiently.

## 🚀 Features
- **Persistent Storage:** Tasks are saved in `localStorage`, so they don't disappear on page refresh.
- **Priority Filtering:** Filter tasks by High, Medium, or Low priority.
- **Task Management:** Add, delete, and toggle completion status of tasks.
- **Responsive Design:** Styled with Tailwind CSS for a clean look on all devices.

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **State Management:** React Hooks (useState, useEffect)

## ⚙️ Setup
1. Clone the repo: `git clone https://github.com/zahrabatool487/smart-task-manager.git`
2. Install dependencies: `npm install`
3. Start the app: `npm start`
